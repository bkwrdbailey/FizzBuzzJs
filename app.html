FIZZ BUZZ: A Coding Project

This program takes in a FIZZ value and a BUZZ value to be used when evaluting the numbers between 1 and 100 to see if they are divisible by the first value given or the second one. The resulting table display will show FIZZ if a number is divisible by the first value, BUZZ if a number is divisible by the second value, FIZZBUZZ if a number is divisible by both values, or just the number itself if it is not divisible by either.
